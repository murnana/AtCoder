# 問題を解くための環境について

`workspaces/`ディレクトリーについて説明します。
このリポジトリでは、AtCoderの実行環境変更に合わせて複数の開発環境を維持しています。

## C# ワークスペース

| ワークスペース | .NET | C# | 対象コンテスト | 状態 |
|----------------|------|----|--------------|----|
| `csharp-dotnet-9-0-8-aot/` | .NET 9.0.8 AOT | C# 13.0 | ABC430以降 (2025/11/01~) | 現行 |
| `csharp-dotnet-7-0-7-aot/` | .NET 7.0.7 AOT | C# 11.0 | ABC429以前 (~2025/10/31) | 旧環境 |

## 環境切り替えのタイムライン

2025年11月1日 (ABC430) を境に、AtCoderのC#実行環境が更新されました:

- **2025/10/31以前 (ABC429まで)**: .NET 7.0.7 AOT / C# 11.0
- **2025/11/01以降 (ABC430から)**: .NET 9.0.8 AOT / C# 13.0

詳細は[AtCoder言語アップデート告知](https://atcoder.jp/posts/1593)を参照してください。

## 主要ライブラリ

各C#ワークスペースには以下のライブラリが含まれています:

- **ac-library-csharp** (3.9.2-atcoder1)
  - AtCoder公式のアルゴリズム・データ構造ライブラリ
  - セグメント木、Union-Find、最大流などの実装を提供

- **SourceExpander** (8.2.0)
  - 複数ファイルのコードを単一ファイルに展開するツール
  - AtCoder提出時に`Combined.csx`として自動生成

- **MathNet.Numerics** (5.0.0)
  - 数値計算ライブラリ
  - 線形代数、統計、最適化などの機能を提供
