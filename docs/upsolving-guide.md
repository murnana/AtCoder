# 復習するためのガイド

## Upsolvingとは

Upsolvingは、コンテスト中に解けなかった問題をコンテスト後に解き直す学習プロセスです。このリポジトリでは、Upsolving解答を専用ディレクトリで管理しています。

## Upsolvingの流れ

1. **問題分析**: コンテスト後、解けなかった問題を分析
2. **公式解説参照**: AtCoderの公式解説や他の参加者の解法を学習
3. **再実装**: 理解した内容を基に、詳細なコメント付きで実装
4. **提出・AC**: AtCoderに提出し、正解を確認
5. **アーカイブ**: `contests/abc###/upsolving/`に解答を保存
6. **README更新**: コンテストREADMEにUpsolving記録を追加

## Upsolving解答の特徴

- **詳細な日本語コメント**: アルゴリズムの説明や考え方を記述
- **計算量分析**: 時間計算量・空間計算量を明記
- **提出URL**: AC提出へのリンクを記録
- **学習ノート**: 問題の本質や学んだテクニックを記述

**例**: [contests/abc434/upsolving/C.cs](../contests/abc434/upsolving/C.cs)
